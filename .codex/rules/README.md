# Codex Rules

Codex-native guardrails and reusable guidance for the Codex runtime.

Use them when:

- Updating `AGENTS.md`
- Validating AI asset changes
- Designing Codex workflows and safeguards

These are reference documents, not implicit runtime hooks. For the corresponding Windsurf rules see `.windsurf/rules/`. For hook intent that translates the Windsurf and plugin-side enforcement into manual review steps see `.codex/operations/hook-intents.md`.
