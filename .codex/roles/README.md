# Codex Role Library

This directory contains 22 Codex-native role references that mirror the Windsurf role rules under `.windsurf/rules/roles/`.

These files are not runtime agent prompts. They are authoring aids for:

- Choosing the right Codex skill stack
- Defining scope boundaries in `AGENTS.md`
- Writing task briefs with the right level of specialization

Each role file captures:

- Mission and ownership
- Default deliverables
- Source-of-truth files
- Preferred Codex skills
- Handoff boundaries

Use these files when adapting plugin-side workflows into Codex instructions or templates, or when reviewing parity between Codex and Windsurf roles.
