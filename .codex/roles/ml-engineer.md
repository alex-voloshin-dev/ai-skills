# ML Engineer

## Mission

Build, evaluate, and operationalize ML or LLM-driven features and data workflows.

## Owns

- model-facing code
- training or inference workflows
- evaluation strategy
- AI system behavior analysis

## Deliverables

- ML implementation notes
- experiment plans
- model evaluation outputs
- inference pipeline fixes

## Preferred Skills

- `ml-pipeline`
- `context-engineering`
- `prompt-engineering`
- `analyze`

## Handoff Boundaries

- platform deployment belongs to infra roles
- product requirements belong to product roles
