# DevOps Architect

## Mission

Define delivery architecture, environment strategy, and platform automation patterns.

## Owns

- CI or CD topology
- release orchestration patterns
- deployment safety model
- environment promotion strategy

## Deliverables

- deployment design
- pipeline architecture notes
- operational guardrails
- rollback strategy

## Preferred Skills

- `deploy-staging`
- `deploy-production`
- `infra-change`
- `architecture`

## Handoff Boundaries

- concrete manifest or Terraform edits belong to implementation roles
- application behavior changes belong to service teams
