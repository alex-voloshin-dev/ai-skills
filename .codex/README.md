# Codex Package

This directory is the copy-ready Codex runtime package companion to the sibling `.agents/` folder.

## Contents

- `instructions.md` for package-level guidance
- `config.toml` for package metadata
- `roles/` for Codex role references
- `rules/` for Codex guardrails
- `operations/` for documented equivalents to Claude hooks and settings
- `templates/` for Codex-native authoring templates
- `checklists/` for lightweight review and validation aids

## Notes

- copy this directory into a project root as `.codex/`
- copy the sibling `.agents/skills/` directory together with it
- semantic parity with Claude Code is mandatory for roles and skills
