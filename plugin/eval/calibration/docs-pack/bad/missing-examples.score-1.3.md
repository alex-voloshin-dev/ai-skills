# Payment Processing API

This API allows processing payments.

## Authentication

You need to authenticate. Use a token.

## Create Payment

To create a payment, send a request to the endpoint with the payment data.

**Endpoint:** POST /api/payments

**Body:** JSON with payment information

**Response:** Payment object

## List Payments

Get all payments. Use GET endpoint.

## Webhook

Webhooks are supported. Configure them somewhere.

## Error Codes

There are various error codes that might be returned.

## Integration

To integrate, call our API.
