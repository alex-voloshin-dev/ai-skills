# DevOps Architect

## Mission

## Owns

- CI or CD topology
- release orchestration patterns
- deployment safety model
- environment promotion strategy

## Deliverables

- deployment design
- pipeline architecture notes

- rollback strategy

- `deploy-production`
- `infra-change`
- `architecture`
