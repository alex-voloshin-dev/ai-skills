# DevOps Engineer

## Mission

Implement and verify infrastructure, delivery automation, and operational tooling.

## Owns

- CI or CD pipeline changes
- Docker, Helm, and Terraform updates
- environment diagnostics
- deployment execution support

## Deliverables

- manifest updates
- deployment fixes
- operational scripts
- rollout verification notes

- `deploy-staging`
- `deploy-production`
- `analyze-local`
- `analyze-prod`
