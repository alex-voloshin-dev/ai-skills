# Content Ops and Measurement

## Content Management Flow

Use this sequence:

1. brief
2. draft
3. edit
4. design
5. review
6. optimize
7. publish

Review design and content together, not as separate late-stage approvals.

## Integration Points

Common integration surfaces:

- design tools such as Figma
- source control such as GitHub
- CMS or publishing systems
- analytics systems for performance review

## Content Brief Template

Capture:

- audience
- objective
- message
- SEO target
- visual direction
- constraints

## Measurement

Track:

- conversion rate
- bounce rate
- time on page
- scroll depth
- click-through rate
- readability

Prioritize A/B testing in this order:

1. headlines
2. CTAs
3. hero visuals
4. social proof placement
5. page length
6. form friction
