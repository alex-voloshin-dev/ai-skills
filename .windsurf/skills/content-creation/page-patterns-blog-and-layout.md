# Page Patterns: Blog, Layout, and Tone

## Blog Post Structure

Recommended order:

1. title
2. metadata
3. introduction
4. body sections
5. key takeaways
6. CTA
7. author bio
8. related posts

## Blog Metadata and SEO

Check project-specific frontmatter in `AGENTS.md`, then confirm:

- title
- description
- author
- publication date
- tags or categories
- cover image
- draft or status
- series metadata when relevant

SEO baseline:

- keyword in title, URL, first paragraph, and meta description
- internal links to at least three existing pages or posts
- descriptive image alt text
- external links only when they add real authority

## Cross-Link and Series Rules

After writing or updating a blog post:

- add outbound links to related internal content
- update existing posts to link back when relevant
- avoid orphaned posts
- verify all series navigation links
- update sitemap, canonical, and structured data surfaces where needed

## Layout-Content Rules

When content arrays drive UI:

- item counts must match grid expectations
- validate symmetry at each responsive breakpoint
- update `AGENTS.md` if content files and rendering components are tightly coupled

## Tone by Page Type

| Page Type | Tone | Energy |
|---|---|---|
| Landing page | Confident, direct | High |
| Homepage | Welcoming, clear | Medium-high |
| Feature page | Specific, informative | Medium |
| Blog post | Helpful, conversational | Medium |
| About page | Warm, authentic | Medium |
| Pricing page | Transparent, reassuring | Low-medium |
| Documentation | Precise, neutral | Low |
