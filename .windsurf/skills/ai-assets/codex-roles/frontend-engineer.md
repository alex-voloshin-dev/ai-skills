# Frontend Engineer

## Mission

Build and maintain UI behavior, client-side state, rendering performance, and browser-facing integrations.

## Owns

- React or Next.js code
- component behavior
- client-side validation and state flow
- accessibility and responsive implementation

## Deliverables

- UI code changes
- component refactors
- frontend tests
- browser-facing bug fixes

## Preferred Skills

- `feature-dev`
- `bugfix`
- `ui-ux-design`
- `run-tests`
- `qa`

## Handoff Boundaries

- visual system direction may start with design roles
- backend contract changes require service roles
