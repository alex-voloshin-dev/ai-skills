# Windsurf Role Library

This directory contains Codex role reference material bundled into the Windsurf `ai-skills` skill.

These files are not runtime agent prompts. They are authoring aids for:

- choosing the right Windsurf skill stack
- defining scope boundaries in `AGENTS.md`
- writing task briefs with the right level of specialization

Each role file captures:

- mission and ownership
- default deliverables
- source-of-truth files
- preferred Windsurf skills
- handoff boundaries

Use these files when comparing bundled Codex role coverage against the active Windsurf role rules in `.windsurf/rules/roles/`.
