# Security Scan

## Workflow

1. Read `AGENTS.md` and the affected service context.
2. Review dependency manifests and configuration files in scope.
3. Look for secrets exposure, unsafe defaults, weak validation, and supply-chain risk.
4. Summarize findings by severity.
5. Recommend concrete remediation steps.
