# MARKETING.md Template

Complete template for marketing strategy initialization. Used by the `/marketing` workflow (Phase 1: Init) to create the `marketing/MARKETING.md` file — the single source of truth for all marketing activities.

## How to Use

1. Copy this template into `marketing/MARKETING.md`
2. Fill in each section based on user interview answers and project context
3. Replace all `[placeholder]` values with actual data
4. Remove sections that don't apply (e.g., paid advertising if budget is zero)
5. Review with user before finalizing

---

## Recommended Tool Stacks by Budget

### Free / Bootstrap

| Need | Tool |
|---|---|
| Analytics | Google Analytics 4, Google Search Console |
| Email | Buttondown (free tier), Resend |
| Social scheduling | Buffer (free tier), manual posting |
| Design | Canva (free), Unsplash (photos), Undraw (illustrations) |
| SEO | Google Search Console, Ubersuggest (free tier) |
| Content | AI assistants (ChatGPT, Claude) for drafts |

### Small Budget ($50-200/month)

| Need | Tool |
|---|---|
| Analytics | GA4 + Mixpanel (free tier) or PostHog |
| Email | ConvertKit, Mailchimp, Resend |
| Social scheduling | Buffer (paid), Typefully (X/Twitter) |
| Design | Canva Pro, Midjourney (images) |
| SEO | Ahrefs Lite or Semrush |
| Content | AI assistants + Jasper or Copy.ai |

### Significant Budget ($500+/month)

| Need | Tool |
|---|---|
| Analytics | Mixpanel or Amplitude + GA4 |
| Email | ConvertKit, ActiveCampaign, Customer.io |
| Social scheduling | Buffer, Hootsuite, Sprout Social |
| Design | Figma + Canva Pro + Midjourney |
| SEO | Ahrefs or Semrush (full) |
| Content | Full AI stack + editorial process |
| Paid ads | Google Ads, LinkedIn Ads |
