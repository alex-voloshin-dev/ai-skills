# Feature Design

## Workflow

1. Clarify the problem and target user.
2. Define the user value and business goal.
3. Write requirements and acceptance criteria.
4. Capture risks, constraints, and out-of-scope items.
5. Prepare a handoff-ready brief for `architecture` or `feature-plan`.
