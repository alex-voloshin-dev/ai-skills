# AI Content Generation Workflows

## Text Generation

Useful tasks:

- headline generation
- value proposition drafts
- feature-to-benefit translation
- FAQ drafting
- social proof formatting

Rules:

- prompt for multiple variants when testing messaging
- specify audience, tone, and constraints
- filter out cliches, vague claims, and inflated wording

## Image Generation

Use AI image tools for:

- hero visuals
- editorial illustrations
- concept variations
- on-brand asset exploration

Rules:

- define subject, style, mood, palette, composition, and aspect ratio
- keep a reusable style reference if consistency matters
- review every output for artifacts, licensing, bias, and brand fit

## Tool Selection

Pick tools by outcome:

- ChatGPT or Claude for long-form drafting and nuance
- Gemini for multimodal synthesis
- Midjourney or DALL-E for fast concept visuals
- Stable Diffusion or Flux when you need more control
- stock libraries or icon libraries when generated assets are unnecessary
