# Analytical Frameworks Reference

Comprehensive reference guide for structured analytical frameworks. Select frameworks based on the question type and decision the analysis should inform.

## Strategic and Business

### SWOT

Use when you need a balanced view of internal strengths and weaknesses versus external opportunities and threats.

### Porter's Five Forces

Use for industry attractiveness, structural competition, and supplier or buyer power.

### Business Model Canvas

Use to map customer segments, value proposition, channels, key activities, and cost structure.

### JTBD

Use to understand the progress the user or buyer is trying to make.

### PESTLE

Use to scan political, economic, social, technological, legal, and environmental forces.

## Systems and Root Cause

### 5 Whys

Use for focused root-cause drilling on a concrete symptom.

### Fishbone

Use to enumerate causal branches before narrowing them down.

### Theory of Constraints

Use when one bottleneck likely governs throughput or progress.

### Cynefin

Use to determine whether the problem is clear, complicated, complex, or chaotic before choosing an approach.

## Decision and Evaluation

### MECE

Use to break the problem into mutually exclusive, collectively exhaustive dimensions.

### Hypothesis-Driven Analysis

Use when speed matters and you need to test plausible explanations quickly.

### Weighted Scoring / MCDA

Use when options must be compared on explicit criteria.

### Cost-Benefit Analysis

Use when tradeoffs can be estimated in comparable units.

### First Principles

Use when inherited assumptions are likely misleading the discussion.

## Technical and Architecture

### Gap Analysis

Use to compare current state versus desired state.

### Architecture Assessment

Use to assess component boundaries, coupling, scalability, and operational risks.

### Technical Debt Assessment

Use to categorize remediation needs across reliability, security, efficiency, and maintainability.
