# Blog Post

Use this workflow for long-form blog content.

## Workflow

1. Read root `AGENTS.md` and blog conventions from the repo.
2. Define the audience, angle, and success criteria.
3. Draft the post with clear structure.
4. Refine title, headings, metadata, and internal links.
5. Verify formatting and repository conventions.

## Output

- Content brief
- Draft or updated article
- SEO and quality notes
